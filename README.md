## Observations
- Browser startup time improved after removal
- Reduced unnecessary background activity
- Improved browsing performance

---

## How Malicious Extensions Harm Users
- Steal browsing data and credentials
- Inject ads or redirect traffic
- Track user activity without consent
- Slow down browser performance
- Act as spyware or malware

---

## Security Best Practices for Browser Extensions
- Install extensions only from official stores
- Review permissions before installing
- Remove unused extensions regularly
- Avoid extensions with poor reviews
- Keep browser updated

---

## Conclusion
This task helped in understanding browser security risks related to extensions. Regular review and removal of unnecessary extensions improves both security and performance.
